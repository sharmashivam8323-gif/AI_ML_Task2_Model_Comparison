# 🏠 AI & Machine Learning - Task 2

## Feature Engineering, Model Optimization & Performance Comparison

### 📌 Objective

The objective of this project is to improve machine learning model performance using feature engineering, data preprocessing, and model comparison techniques.

---

## 📊 Dataset Used

**California Housing Dataset**

Target Variable:

* HousePrice (Median House Value)

Features:

* Median Income
* House Age
* Average Rooms
* Average Bedrooms
* Population
* Households
* Latitude
* Longitude
